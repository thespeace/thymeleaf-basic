SpringEL 표현식

(변수 표현식 : ${...})
그리고 이 변수 표현식에는 스프링 EL이라는 스프링이 제공하는 표현식을 사용할 수 있다.
    Object
  • ${user.username} =
  • ${user['username']} =
  • ${user.getUsername()} =
    List
  • ${users[0].username} =
  • ${users[0]['username']} =
  • ${users[0].getUsername()} =
    Map
  • ${userMap['userA'].username} =
  • ${userMap['userA']['username']} =
  • ${userMap['userA'].getUsername()} =
